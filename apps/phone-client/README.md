# phone-client

## Building

Run `nx build phone-client` to build the app as a library to be used by game-command-center.

Run `nx build phone-client --mode app` to build the app as an app.

## Running Locally

Run `nx dev phone-client` to run the app locally.

## Deploying

This project uses [Vercel](https://vercel.com/), a cloud platform that makes it easy to deploy
web applications very easily.
